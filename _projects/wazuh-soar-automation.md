A Docker-based SIEM and SOAR platform integrating Wazuh Manager, OpenSearch, and a web dashboard for centralized security monitoring. It provides automated threat detection, host monitoring via Wazuh agents, and a turnkey deployment workflow with pre-configured containers and encrypted communications.

**Key Features:**
- Integrated Wazuh Manager and OpenSearch stack in Docker
- Automated threat detection and active response playbooks
- Host monitoring via lightweight Wazuh agents
- Centralized web dashboard for security event visualization
- Turnkey deployment with pre-configured containers
- Encrypted agent-to-manager communications
